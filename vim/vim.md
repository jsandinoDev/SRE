# Vim Noob Guide

### Modes

```
Insert Mode
:i

Visual Mode
:v

Exit modes
Esq:

```

### Exit/Save/Interface

```
Exit file without save
:q!

Save
:w
:Shift + ZZ

Save and exit
:wq!
:x
:Shift + ZQ

Add numbers to each line
:set nu

Remove numbers
:set nu!
```

### Common commands

```
Delete line
:dd

Paste
:p

Copy current line
:yy

Create a new line an insert mode
:o

Go back
:u

Go next
:Control + r

Search
:/onclick -> n --> shift + n

Go top file
:gg

Go end file
:G
```
